Cluster genomic regions based on somatic CNV counts¶

  • Objective: Use k-means clustering and t-SNE to identify genomic regions that are enriched with or containing similar counts of somatic gains and losses
  • Source file: 50kb genomic regions with the log2 ratio of somatic CNV count / germline CNV counts
  • Steps involved:
    1. Run Circular Binary Segmentation (CBS) to segment the source file (Run CBS to merge genome-wide 50kb genomic regions into consecutive segments with similar log2 ratios)
    2. Extract CBS segments with high somatic CNVs (Mean log2 ratio of somatic CNV count / germline CNV counts > 2)
    3. CBS segments with high somatic CNVs are considered as sCNV-segments
    4. Calculate the number of somatic gains and losses in each sCNV-segments
    5. Cluster sCNV-segments using K-means clustering and visualise clusters using t-SNE

def overlap(q_st, q_end, res_st, res_end):
    # Overlap percentage relative to the query
    o  = min(q_end, res_end)-max(q_st, res_st)
    return o

## Count the number of somatic loss and gain in each input region

def cnv_ann(a, colID, loss, gain, tabix_file): #Manually curated DGV
    tot_losses = 0
    tot_gains = 0

    try:
        for gene in tabix_file.fetch("chr{}".format(a['chrom']), int(a['start']), int(a['end'])):
            #print (gene)
            if gene:
                g = gene.split('\t')
                overlap_len = overlap(int(a['start']), int(a['end']), int(g[1]), int(g[2]))
                if g[colID] == loss:
                    if overlap_len/float(int(a['end']) - int(a['start'])) > 0.5: # 50% overlap with the target CNV
                        tot_losses += 1
                if g[colID] == gain:
                    if overlap_len/float(int(a['end']) - int(a['start'])) > 0.5: # 50% overlap with the target CNV
                        tot_gains += 1
        if tot_losses>0 or tot_gains > 0:
            return tot_losses+1, tot_gains+1 # add 1 to each count; help calculate the ratio of tot_losses/tot_gains in downstream step
        else:
            return 1, 1
    except ValueError:
        return 1, 1
